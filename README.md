Browser Extensions Security Review

Description

This repository contains a security review and documentation of browser extensions installed on Google Chrome and Opera browsers. The purpose of this task is to identify installed extensions, analyze their permissions, check for suspicious or malicious behavior, and document the actions taken to improve browser security and performance.


---

Objective

Review all installed browser extensions

Identify unused or suspicious extensions

Understand risks posed by malicious browser extensions

Document actions taken and final security status



---

Browsers Reviewed

Google Chrome

Opera Browser

Extensions Summary

Google Chrome

Google Docs Offline – Safe and trusted (official Google extension)

Microsoft Bing Search for Chrome – Safe but optional (changes default search engine)


Opera

Opera Ad Blocker – Safe and recommended (official Opera extension)

Opera Wallet – Safe but optional (crypto wallet feature)

Risks of Malicious Extensions

Malicious browser extensions can:

Steal sensitive data such as passwords

Track user browsing activity

Inject unwanted advertisements

Redirect users to malicious websites

Slow down browser and system performance
